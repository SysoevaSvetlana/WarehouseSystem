# Складской учет - Frontend

фронтенд для системы складского учета.

## Технологии

- React 18
- React Router 6
- Axios
- Vite

## Установка и запуск

### 1. Установка зависимостей

```bash
cd frontend
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

### 3. Сборка для продакшена

```bash
npm run build
```

## Функциональность

### Аутентификация
- Вход в систему (JWT токены)
- Автоматическое перенаправление при отсутствии авторизации

### Разделы

1. **Shipments (Поставки)**
   - Просмотр всех поставок
   - Создание поставок (приход/списание)
   - Добавление товаров в поставку

2. **Warehouses (Склады)**
   - Просмотр складов
   - Создание новых складов
   - Удаление складов (только ADMIN)

3. **Stock (Остатки)**
   - Просмотр остатков на складах
   - Фильтрация по товару и складу

4. **Products (Товары)**
   - Просмотр товаров
   - Создание новых товаров
   - Удаление товаров

5. **Users (Пользователи)**
   - Просмотр пользователей (только ADMIN)
   - Изменение ролей (только ADMIN)
   - Удаление пользователей (только ADMIN)

## API Endpoints

Фронтенд работает с следующими эндпоинтами бэкенда:

- `POST /api/auth/sign-in` - Вход
- `POST /api/auth/sign-up` - Регистрация
- `GET /api/shipments` - Список поставок
- `POST /api/shipments/incoming` - Создать приход
- `POST /api/shipments/write-off` - Создать списание
- `GET /api/warehouses` - Список складов
- `POST /api/warehouses` - Создать склад
- `GET /api/stock` - Остатки
- `GET /api/products` - Список товаров
- `POST /api/products` - Создать товар
- `GET /api/users` - Список пользователей
- `PATCH /api/users/{id}/role` - Изменить роль

## Роли пользователей

- **ROLE_ADMIN** - Полный доступ ко всем функциям
- **ROLE_STOREKEEPER** - Доступ к операциям с товарами и поставками

## Конфигурация

Прокси для API настроен в `vite.config.js`:
- Все запросы к `/api/*` перенаправляются на `http://localhost:8080`

## Примечания

- Токен JWT хранится в localStorage
- При ошибке 401 происходит автоматический выход из системы
- Все формы имеют базовую валидацию
